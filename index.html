Who I am.

Designer & Developer based in Bangkok City.

Hello, my name is Aung Kyaw Kyaw a.k.a Alax. I'm originally from Yangon, Myanmar. I started having my strong interest in technology and web development back in 2014 when I was developing several web pages with "Dreamweaver CC 2014" and manipulating data storages with MySQL open source database. That taught me much about HTML, CSS.

Fast-forward to today, I've trained and worked for 1000+ hours on projects and courses with HTML5, CSS3/SASS, Data Structures and Fundamentals of JavaScript. Eager to engage with web development/design challenges to sharpen skills for better user experiences. I've also trained and work on some projects with famous JavaScript libraries and frameworks like Bootstrap, jQuery and amazing React.js. Recently, I have built several projects with those popular Front-End Development libraries that gave me essential knowledge to collaborate and work with future tech teams.

My main objectives and focus these days are creating accessible and functional digital experiences for variety of clients.
