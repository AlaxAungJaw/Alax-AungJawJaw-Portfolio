Random Quotes Machines


by using CSS, JavaScript (Babel), Bootstrap and React.js

Quotes generator app that displays various of quotes with authors of each quote once click the button,
and dynamically changes background images once reload.

See the Pen Random Quotes Machines (freeCodeCamp) by Aung Kyaw Kyaw Alax (@aung-kyaw-kyaw-alax) on CodePen.

  • In the quote box, the text, the author, "Get Quote" & "Tweet" buttons are organized with HTML, then, wrapped and displayed using CSS properties.
  • The quotes data are fetched via an API url link:('https://gist.githubusercontent.com/camperbot/5a022b72e96c4c9585c32bf6a75f62d9/raw/e3c6895ce42069f0ee7e991229064f167fe8ccdc/quotes.json') which is in json data format.
  • On the first load, a random quote will already be displayed in text area.
  • The quote machine app will fetch the data once user clicks, it will fetch new quote with new author and will be displayed.
  • As a feature, a href for "twitter.com/intent/tweet" path has linked as a button and user can able to tweet for their favorite quote that currently displayed.
